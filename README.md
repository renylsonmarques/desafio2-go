# desafio2-go

Este repositório contém a solução do desafio da DIO no curso de Go.

O desafio consiste em implementar o jogo "PinPan", que imprime os números de 1 a 100 e substitui:
- múltiplos de 3 por `Pin`
- múltiplos de 5 por `Pan`
- múltiplos de 3 e 5 por `PinPan`

O arquivo principal da solução é `pinpan.go`.
